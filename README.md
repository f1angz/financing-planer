# Планировщик финансов

Десктопное приложение для учёта личных финансов, разработанное на JavaFX.

## Возможности

- 📊 **Обзор финансов** - визуализация доходов и расходов с помощью круговых диаграмм
- 📁 **Управление категориями** - создание и редактирование категорий доходов и расходов
- 📝 **История операций** - просмотр последних финансовых транзакций
- 🎨 **Современный UI** - тёмная тема с приятным дизайном

## Технологии

- Java 23
- JavaFX 21.0.1
- Maven
- SQLite (с возможностью миграции на PostgreSQL)
- JDBC + HikariCP (connection pool)

## Запуск приложения

### Через Maven

```bash
mvn clean javafx:run
```

### Через командную строку (после сборки)

```bash
mvn clean package
java -jar target/financing-planer-1.0-SNAPSHOT.jar
```

## Основные функции

### Главная страница
- Отображение круговых диаграмм доходов и расходов
- Фильтрация по периодам (год/месяц/день)
- Список последних операций

### Управление категориями
- Добавление новых категорий
- Выбор цвета для категории
- Удаление категорий

### База данных
- Все данные сохраняются в SQLite БД
- Возможность миграции на PostgreSQL (см. MIGRATION_GUIDE.md)
- Автоматическая инициализация схемы БД
- Connection pooling для эффективной работы
